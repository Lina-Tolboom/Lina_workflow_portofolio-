# Lina_workflow_portofolio
Dit is het portofolio van Lina Tolboom

# Portfolio repository

Deze repository is georganiseerd volgens de Guerilla Analytics principes voor data management.

## Structuur

- index.Rmd: startpagina van de portfolio website
- 01_cv_en_toekomstplannen.Rmd: CV pagina en toekomstbeeld
- 02_vrije_opdracht.Rmd: vrije opdracht beschreven
- 03_data_management.Rmd: pagina over data management
- 04_reproducible_science.Rmd: pagina over de reproduceerbare opdracht
- 05_sql.Rmd: Toets gehaald
- 06_R_package.Rmd: pagina over gemaakte R package
- 07_Introductie_projecticum.Rmd: een introductiepagina over het project dat ik ga doen
- 08_geparametrizeerde_Rmarkdown.Rmd: pagina over de parametrizeerende opdracht
- 09_referenties.Rmd: pagina voor de gebruikte referenties van pagina 7
- _bookdown.yml en _output.yml: configuratiebestanden voor bookdown
- _book/: gebouwde website output
- data/: data-gerelateerde mappen en bestanden
- results/: outputbestanden zoals html-bestanden
- docs/: documentatie
- README.md: beschrijving van de repository

## Toelichting

De bookdown bronbestanden staan in de hoofdmap omdat bookdown deze daar nodig heeft. 
Outputbestanden zijn ondergebracht in de map `results`, en data-gerelateerde bestanden in de map `data`.
